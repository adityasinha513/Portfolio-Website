## 💻 Getting Started

To run this portfolio locally:

1. Clone the repository
   ```sh
   git clone https://github.com/adityasinha513/Portfolio-Website.git
   ```

2. Install dependencies
   ```sh
   npm install
   ```

3. Start the development server
   ```sh
   npm start
   ```

4. Deploy to GitHub Pages
   ```sh
   npm run deploy
   ```
